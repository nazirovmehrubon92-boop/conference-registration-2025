# Регистрация участников конференции

Учебный проект по веб-разработке (HTML + CSS + JavaScript + PHP + MySQL)

## Как запустить

1. Создайте базу данных `conference`
2. Импортируйте таблицу из SQL-кода выше
3. Настройте параметры подключения в `config.php`
4. Разместите проект на веб-сервере с PHP 8+ и MySQL (рекомендуется OpenServer / XAMPP / MAMP)

## Функционал

- Адаптивная форма регистрации
- Клиентская + серверная валидация
- Сохранение данных в MySQL
- Страница подтверждения регистрации
- Таблица со всеми участниками
